### アイテム [小型爆弾] の実装
 - [ ] エメラルド 4個
 - [ ] 狂人 / 人狼限定アイテム
 - [ ] 人にぶつけることで仕掛けられる
 - [ ] 起爆装置を投げることで3秒後に爆発
 - [ ] 解除要素あり（まだ決まってない）
### アイテム [メガホン] の実装
 - [ ] エメラルド 3個
 - [ ] 誰でも購入可能
 - [ ] 投げることで発動
 - [ ] 投げて、一回だけは距離関係なくテキストチャットが送ることができる\
※ メガホンは近くでしか人と喋ることができない時のみ購入可能
 - [ ] 発動から30秒以内に送らないと効力を失う
 - [ ] 発動時は他のプレイヤーはメガホンを使えなくなる
 - [ ] 効力を失わず送信された場合は、送信されてから20秒間は **全員** メガホンを使えなくなる\
※ 効力を失った場合は誰でも即座に利用できる
