# ラブスペ人狼
**このゲームは開発中です**

# ルール説明
このゲームは人狼ゲームに脱出要素を加えてスパイスをかけたようなゲームです

## 役職
- 人狼（黒）
勝利条件は白側の全滅（探偵は倒さなくても良い）／白側の過半数の死亡
- 狂人（黒）
勝利条件は人狼と同じ
- 探偵（白）
勝利条件は黒側の全滅（狂人は倒さなくても良い）／黒側の過半数の死亡
- 目撃者（白）
勝利条件は探偵と同じ
- 市民（白）
勝利条件は探偵と同じ

## システムについて
### アイテムの簡単な説明
- 特殊アイテム
  
役職限定／全員利用可能と分けられています。\
アイテムによっては **[投]** と説明欄に記載されています。このアイテムは投げることによって利用できます。
- チェストアイテム

**エンダーチェスト** から集められるアイテムです。詳しくは「エンダーチェスト」の項目にて

集めたチェストアイテムはマップ内に設置されている村人と物々交換ができます。
- エメラルド
- 脱出アイテムと交換できるアイテム
- ゴミ

がチェストから手に入ります。

### 村人について
マップ上に、村人が設置されています。
- 特殊アイテム
- 脱出アイテム
- 怪しい村人

集めたアイテムは、「特殊アイテム」・「脱出アイテム」（脱出のために使うアイテム）と交換できます。\
特殊アイテムは全てエメラルド、「脱出アイテム」はチェストから集めた「脱出アイテムと交換できるアイテム」と交換できます。\
また、「怪しい村人」にゴミを3個以上集めて交換することで、交換したゴミの数相応のエメラルドがもらえます。

### エンダーチェスト
マップ内に **エンダーチェスト** が散らばっており、「**チェストの鍵**」を使ってチェストからアイテムを取ることができます。

チェストからは、
- エメラルド
- 脱出交換アイテム
- ゴミ
  
が集められます。

鍵は一度使うとなくなりますが、5秒後に自動的に手元に入ります。\
ですが確率で鍵を落としてしまうことがあります。その時は「特殊アイテム」の村人からエメラルド2個で鍵と交換してもらいましょう。

> [!NOTE]
> 鍵は所持しているエメラルドが4個以上のときのみ落とします。

### 攻撃方法
全プレイヤーはお互いを殴ることができますが、ダメージは入りません。\
人狼は、人狼限定の特殊アイテムを使うことでダメージを与えることができます。

- 探偵は3回
- 目撃者は1回
- 市民は2回（狂人も2回）

役職によって上記の回数、ダメージを与えることで死にます。（**例外あり**）
> [!IMPORTANT]
> 人狼は、目撃者を倒すと凶暴化します。
> 人狼は目撃者を探すことを優先することをお勧めします。

> [!NOTE]
> ダメージを与えても与えられた側はハートが減りませんが、きちんとカウントされています。

### 脱出方法
ゲームが開始した際に、市民（目撃者を含む）にチャットで脱出に必要なアイテムを二つお知らせされます。\
この脱出アイテムは、村人から「脱出効果アイテム」と交換することで手に入れることができます。

> [!IMPORTANT]
> 目撃者が脱出してしまうと、倒された時と同様に人狼が凶暴化します。

## アイテムについて

### 注意してほしいこと
矢を除いた特殊アイテム全部がそうなんですが、**利用するとインベントリに残してる同じアイテムも全部消えます。**
> 例えば ... おぼろ灯を使ったらインベントリ内に隠してたおぼろ灯も全部消えた

なので、購入時は重複購入をしないように気をつけてください。

### 特殊アイテム（黒側）
黒側限定アイテムは、白側が購入することができません。\
購入するとアイテムが没収され、アイテム1個分のエメラルドが返金されます。
- ナイフ（狂人は購入不可）\
市民にダメージを与えるためのアイテムです。\
一度市民をナイフで刺すと透明 + 移動速度低下が3秒かかり7秒間ナイフが没収されます。\
市民には盲目が5秒間かかります。

- 汚れたナイフ\
目撃者を倒して凶暴化した際に手に入るナイフです。\
**通常のナイフ時の制限が一切無くなり、ダメージを与えた回数に関係なく一撃で倒せるようになります。**

- 小型爆弾（狂人は購入不可）
> [!WARNING]
> こちらは開発中のため利用できません。

- おぼろ灯（とう）（**狂人も購入可能**） [投] \
使うと、20%の確率で灯りが目撃者が誰か教えてくれます。

- ナイフ（レプリカ）（**狂人も購入可能**）\
ナイフのレプリカです。通常のナイフよりも格安ですが、ダメージを与えることはできない玩具です。

### 特殊アイテム（探偵）※
- 弓\
人狼に打ち勝つ唯一のアイテムです。\
購入すれば矢（別売り）がある限り撃つことができますが、プレイヤーに当たったら壊れます。\
どんな役職も一撃で倒すことができます。

**実は...**\
探偵用アイテムと記載していますが、**狂人は最初から弓を買うことができます。**

> [!IMPORTANT]
> 探偵が倒されるなどで居なくなると全役職が弓を買うことができるようになります。

### 特殊アイテム（共通）※
- 矢\
矢は誰でも買えます

> [!NOTE]
> 一度矢がプレイヤーに当たると問答無用でインベントリ内の矢が全部没収されます。

- 包帯（羽）[投] \
人狼に負われたダメージを回復することができます。**1回まで利用可能、目撃者は購入はできますが、利用できません**
- インシデント・レポート（名札）[投] \
使うと、現時点で誰が倒されたか知ることができます。
- 守護のお守り（鉄の馬鎧）[投] \
使うと、10秒間無敵になる（ダメージが入らない）代わりに脱出や他アイテムの使用など**全てができなくなります。**
- 神秘的なお守り（不死のトーテム）\
**次のダメージで死ぬ状態かつ手に持った状態（メイン・オフ関係なく）で人狼から攻撃される**ことで死を免れることができます。

> [!IMPORTANT]
> 弓によるダメージには効果を発揮しません。
> また、まだ次のダメージで死ぬことがない状態で手に持ち、攻撃されるとお守りが壊れます。

- 神秘的なお守り（不死のトーテム）（レプリカ）\
神秘的なお守りのレプリカ。持っていても効果を発揮することはありません。

- 鈍化のスプラッシュポーション（2秒）
- 透明化のポーション（15秒）
- 盲目の残留ポーション（3秒）

自分で飲むポーションは、持ちながらシフトを押すことで即座に飲むことができますが、大きなゲップをします。
