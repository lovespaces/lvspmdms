# ラブスペ人狼
**このゲームは開発中です**

# ルール説明
このゲームは人狼ゲームに脱出要素を加えてスパイスをかけたようなゲームです

## TL;DR
- 白側（探偵・**目撃者**・市民）は「人狼を全滅」させるか、「白側の必要人数（半数以上）が脱出」すると勝利
- 黒側（人狼・狂人）は、白側を半数未満（人狼と同数以下）に排除すると勝利
- 白側はチェストからアイテムを集め、村人と交換して脱出アイテムを揃える
- 黒側は村人と交換をして「ナイフ」などのアイテムを入手する
- 脱出条件を満たしたら、金ブロックの看板前でしゃがみ続けることで脱出可能
- 人狼は特殊アイテム「ナイフ」でプレイヤーを攻撃できる
- **[重要]** 目撃者が死亡・誤射・条件未達での脱出・途中抜けをすると、人狼が「凶暴化」して非常に危険になる
- 探偵は弓で誰でも一撃で倒せるが、誤射に注意
- 死亡または脱出すると観戦者になる

## 役職
| 役職 | 陣営 | 勝利条件 |
|-----|-----|-----|
| 探偵 | 白 | 人狼の排除、もしくは白側の半分以上の脱出 |
| 目撃者 | 白 | 探偵と同じ（※**生存が超重要**) |
| 市民 | 白 | 探偵と同じ |
| 人狼 | 黒 | 白側の半数以上を狩る |
| 狂人 | 黒 | 人狼と同じ |

> [!NOTE]
> 人数計算の際、同数の場合は白側の勝利が優先されます。

生存中の白プレイヤー（探偵は含まない）が必要人数以上が脱出するか、人狼がいなくなった時点でゲームが終了します。 \
ここでの必要人数は「ゲーム開始時点の白側人数」の半数以上を表しています。

> [!IMPORTANT]
> **目撃者の生存によってゲームの流れが大きく変わります。**\
> 目撃者は慎重に動くことを心がけましょう

## システムについて
### アイテムの簡単な説明
- 特殊アイテム
  
役職限定／全員利用可能と分けられています。\
アイテムによっては **[投]** と説明欄に記載されています。このアイテムは投げることによって利用できます。
- チェストアイテム

**エンダーチェスト** から集められるアイテムです。詳しくは「エンダーチェスト」の項目にて

集めたチェストアイテムはマップ内に設置されている村人と物々交換ができます。

### 村人について
マップ上に、村人が設置されています。
- 特殊アイテム
- 脱出アイテム
- 怪しい村人

集めたアイテムは、「特殊アイテム」・「脱出アイテム」（脱出のために使うアイテム）と交換できます。\
また、「怪しい村人」にチェストから出てくるゴミを3個以上集めて交換することで、交換したゴミの数相応のエメラルドがもらえます。

### エンダーチェスト
マップ内に **エンダーチェスト** が散らばっており、「**チェストの鍵**」を使ってチェストからアイテムを取ることができます。

鍵は一度使うとなくなりますが、5秒後に自動的に手元に入ります。\
ですが確率で鍵を落としてしまうことがあります。その時は「特殊アイテム」の村人から鍵をエメラルドで交換できます。

> [!NOTE]
> 鍵は所持しているエメラルドが4個以上のときのみ落とします。

### 攻撃方法
全プレイヤーはお互いを殴ることができますが、ダメージは入りません。\
人狼は、人狼限定の特殊アイテムを使うことでダメージを与えることができます。

- 探偵は3回
- 目撃者は1回
- 市民は2回（狂人も2回）

役職によって上記の回数、ダメージを与えることで死にます。（**例外あり**）

> [!NOTE]
> ダメージを与えても与えられた側はハートが減りませんが、きちんとカウントされています。

### 凶暴化について
ゲーム中、特定の条件を満たすと、人狼が凶暴化状態に突入します。\
凶暴化すると、人狼の戦闘能力が圧倒的に強化され、白側は全滅の危機に陥ります。
> [!NOTE]
> 狂人は凶暴化しません

**凶暴化のトリガー（発生条件）**
以下のいずれかが発生した瞬間、人狼が凶暴化します。

- 人狼が目撃者を倒す
- 白側が誤射で目撃者を倒してしまう
- 目撃者が、必要人数に達していない状態で脱出する
- 目撃者がゲーム中に途中抜けする

**凶暴化時の強化内容**
凶暴化すると、人狼に専用武器が支給され、凶暴化以前に持っていた制限が緩和され、人間狩りがしやすくなります。\
詳しくは「アイテムについて」における「汚れたナイフ」の項目をご覧ください。

### 脱出方法
ゲームが開始した際に、市民（目撃者を含む）にチャットで脱出に必要なアイテムを二つお知らせされます。\
この脱出アイテムは、村人から「脱出効果アイテム」と交換することで手に入れることができます。

脱出アイテムを集めたら、`[脱出する]` と書かれた看板がある金ブロックの上で**6秒以上しゃがんでください**。\
しゃがみに成功するとチャットに「脱出できるようになった！」と表示されるので、しゃがんだまま**何も持たずに**看板を右クリックしてください。

> [!IMPORTANT]
> 前述の通り、目撃者は、必要人数に達していない状態（画面に表示されます）で脱出をすると人狼が凶暴化してしまいます。

### 誤射について
白側が、間違えて市民や目撃者を倒してしまうと、「誤射」となります。\
誤射時は、その時点ではプレイヤーに影響はありませんが、目撃者を誤射してしまった場合は人狼が凶暴化し、人狼の勝利への手助けにもなってしまうのでお気をつけください。

### 観戦者について
ゲーム内で死亡もしくは脱出をすると、スペクテイターになり観戦者となります。\
観戦者は、生き残っているプレイヤーたちとVCを行っている場合は**必ずミュートにしてください**。\
なお、**観戦者が送るチャットは全て観戦者のみ閲覧することができます。生存者からは見ることができません。**

### 途中抜け・途中参加について
ゲーム中に途中抜けしてしまった場合、（観戦者の途中抜けはカウントされない）死亡判定となりゲームが進みます。\
また前述した通り、目撃者が途中抜けした場合、**途中抜けした時点で人狼が凶暴化します**のでご注意ください。

ゲーム中にサーバーに途中参加したプレイヤーは自動的に観戦者となります。

### 近距離モードについて
> [!NOTE]
> 制作者である lovespaces による運営時のみこのモードが遊べます

このゲームでは、「通常モード」と「近距離モード」があります。\
近距離モードとは、近くのプレイヤー（**4マス以内**）とだけボイス・テキストでの会話が可能になります。

## アイテムについて

### 注意してほしいこと
矢を除くすべての特殊アイテムは、**使用するとインベントリ内にある「同じアイテム」がすべて連動して消滅します**。
> 例えば ... 包帯を使ったらインベントリ内に隠していた包帯も全て消えます。\
重複分が消えたとしても、効果は一度しか発揮されません。

そのため、ショップでの重複購入（まとめ買い）はしないよう気をつけてください。

### 特殊アイテム（黒側）
黒側限定アイテムは、白側が購入することができません。\
購入するとアイテムが没収され、アイテム1個分のエメラルドが返金されます。
- **ナイフ（狂人は購入不可）**\
市民にダメージを与えるためのアイテムです。\
一度市民をナイフで刺すと透明 + 移動速度低下が3秒かかり7秒間ナイフが没収されます。\
市民には盲目が5秒間かかります。

- **汚れたナイフ**\
目撃者を倒して凶暴化した際に手に入るナイフです。\
**通常のナイフ時に付与されていた透明化がなくなりますが、ダメージを与えた回数に関係なく一撃で倒せるようになります。**\
また、ナイフの没収時間も3秒に縮小されます。

- **小型爆弾（狂人は購入不可）**
> [!WARNING]
> こちらは開発中のため利用できません。

- **おぼろ灯（とう）（狂人も購入可能） [投]** \
使うと、20%の確率で灯りが目撃者が誰か教えてくれます。

- **ナイフ（レプリカ）（狂人も購入可能）**\
ナイフのレプリカです。通常のナイフよりも格安ですが、ダメージを与えることはできない玩具です。

### 特殊アイテム（探偵）※
- **弓**\
人狼に打ち勝つ唯一のアイテムです。\
購入すれば矢（別売り）がある限り撃つことができますが、プレイヤーに当たったら壊れます。\
どんな役職も一撃で倒すことができます。

**実は...**\
探偵用アイテムと記載していますが、**狂人は最初から弓を買うことができます。**

> [!IMPORTANT]
> 探偵が倒されるなどで居なくなると全役職が弓を買うことができるようになります。

- **矢**\
弓を使うために必要なアイテム。\
**狂人は弓は買えますが、探偵が居なくなるまで矢を買うことは出来ません。**

> [!NOTE]
> 一度矢がプレイヤーに当たると問答無用でインベントリ内の矢が全部没収されます。

> [!IMPORTANT]
> 探偵以外のプレイヤーは一回のみ他プレイヤーに当てることができます。

### 特殊アイテム（共通）※

- **包帯（羽）[投]** \
人狼に受けたダメージを回復することができます。**1回まで利用可能、目撃者は購入はできますが、利用できません**
- **インシデント・レポート（名札）[投]** \
使うと、現時点で誰が倒されたか知ることができます。
- **メガホン (ホッパー) [投]**
> [!IMPORTANT]
> このアイテムは、「近距離モード」時のみ利用ができます。

使うと、一度だけ全員に向けてテキストチャットを送ることができます。\
ですが、30秒以内に送らないと効力が発揮されないのでご注意ください。\
チャットが送信されると、送信者を含めた全プレイヤーは20秒間、メガホンが使えなくなります。
- **守護のお守り（鉄の馬鎧）[投]** \
使うと、10秒間無敵になる（ダメージが入らない）代わりに脱出や他アイテムの使用など**全てができなくなります。**
- **神秘的なお守り（不死のトーテム）**\
人狼の攻撃から自分を守ってくれます。\
どちらかの手で持ち、ダメージを受けることで効果を発揮します。

> [!IMPORTANT]
> 「まだ死なない体力」の時に手に持った状態で攻撃を受けると、効果を発動せずにお守りが壊れてしまうため、持つタイミングに注意してください。\
> また、**弓によるダメージには効果を発揮しません**。

- **神秘的なお守り（不死のトーテム）（レプリカ）**\
神秘的なお守りのレプリカ。持っていても効果を発揮することはありません。

- **鈍化のスプラッシュポーション（2秒）**
- **透明化のポーション（15秒）**
- **盲目の残留ポーション（3秒）**

自分で飲むポーションは、持ちながらシフトを押すことで即座に飲むことができますが、大きなゲップをします。
